102 etiquetas de HTML

  • html

    Significado: Lenguaje utilizado para crear páginas web.
    Uso: Estructurar y definir contenido en una página web utilizando etiquetas y elementos.
    Ejemplo:
    < !DOCTYPE html>
    < html> Mi pagina web< /html>

  • head

    Significado: Sección de un documento HTML que contiene metadatos y enlaces a recursos.
    Uso: Definir información relacionada con el documento como título, codificación de caracteres y enlaces a hojas de estilo y scripts.
    Ejemplpo:
    < html> Mi pagina web< /html>
    < head>
    < h1> TITULO < /h1>
    < /head>

  • !DOCTYPE

    Significado: Lenguaje utilizado para crear páginas web.
    Uso: Estructurar y definir contenido en una página web utilizando etiquetas y elementos.
    Ejemplo de uso:
    < !DOCTYPE html>
